win10预览版14251更新内容汇总
作者：佚名 时间：2016-05-16
一直在不断进行修复完善提升的win10系统，这次推出的win10预览版14251又有什么改变呢?下文小编就为大家奉上win10预览版14251更新内容汇总，想知道win10预览版14251更新了什么内容的朋友一起跟小编去下文看看吧。

今日凌晨，微软向 Windows Insider 用户推送了 Windows 10 Redstone 第四个预览版本更新——Windows 10 Build 14251 预览版更新。虽然版本号直接从 Build 11102“飞跃”至 Build 14251。但主要是修复 BUG 及底层改进，并无重大更新。而且一些已经问题在此前版本中就已经存在，此次并未成功修复，稳定性不足，建议大家谨慎升级。

从 2015 年 12 月 17 日凌晨推送 Redstone (红石) 首个预览版本 Build 11082 更新，到今日的 Build 14251，两个月多月时间完成四次更新，且几个版本之间并无明显区别，说实话，微软这个速度真是不快，甚至是行动缓慢了。不过，微软表示，他们正在酝酿一个大招，下一个预览版会有一个重大功能性升级。但官方并未公布具体推送时间。至于大家更关心的 Windows 10 Mobile 正式版何时推送，微软方面也未给出明确说法。

值得一提的是，对于此次预览版版本号突变，微软方面解释称，以前 Windows PC 版和移动版是完全不同的系统，采用不同的版本号，但是 Windows 10 采用了“大一统”战略，为将 Windows 10 PC 端和移动端统一起来，微软将采用统一的版本号。目前的情况是移动端版本号领先 PC 端太多，所以只能让 PC 端大幅度跳跃一下，直接突变至 Build 14251。

以下是 Build 14251 升级内容：

• 修复了 Windows 图形堆栈中的一处 Bug，该 Bug 导致某些游戏在切换到全屏、改变分辨率或启动时出现崩溃。受影响游戏包括巫师 3、辐射 4、古墓丽影系列、刺客信条系列、合金装备 V 等。

• 修复了某些屏幕阅读类应用软件间歇性崩溃问题，受影响应用包括内置讲述人和放大镜及其他第三方辅助工具。

• 修复了 DPI 设置为 175%时，文件资源管理器频繁崩溃的问题。
