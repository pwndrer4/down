三星GALAXY Note 6最新消息!三星GALAXY Note 6将推出低配版
作者：佚名 时间：2016-05-20

很多人都在期待GALAXY Note 6的上市从目前传出的消息来看三星Note 6将搭载5.8英寸屏幕(确切地说是5.77寸)，比现在Note 5的5.7英寸稍大了一些，而分辨率还是2560x1440像素的2K级别。另外在处理器方面，除了骁龙823(820的升级版之外)，理论上还会有新款Exynos(Exynos 8890的升级版)。机身内存据说还会达到惊人的256g;但是大部分商业人士毕竟还是用不到如此配置;那么最近就有传三星GALAXY Note 6将推出低配省电版详情请看下文。

近日，有网友在微博上爆料称，三星GALAXY Note 6将有低配省电版推出，并会借助降低触控屏分辨率，采用低频版处理器来获得更持续的电池续航表现。

以往提升智能手机的电池续航时间，通常是借助省电模式和增大电池容量来实现，但三星显然还找到了更多的办法，这便是从硬件配置上下手，通过删减“耗电大户”的规格来获得更好的续航表现。按照网友@Rec2020_Eight在微博上的爆料称，三星GALAXY Note 6将会有低配省电版推出，主要变化是将5.8英寸触控屏的分辨率降至1080p，并采用了省电的低频版骁龙820处理器，辅以4GB RAM的内存容量，这样结合配备的大容量电池以及其他相同的配置，自然相比高配版能够得到更理想的待机时间。

虽然这样的说法尚未得到证实，但由于三星GAAXY Note 6主要的是面向商务用户，而电池的续航表现自然也是目标群体所十分关注的内容。所以三星这样的做法无疑十分的精明，一方面低配省电版有着更好的续航表现，一方面即便在配置上有所缩水，但在性能上同样与其他同档次机型不遑多让，这对于商务用户而言同样颇具吸引力。

当然，对于那些技术控和极客们而言，三星GALAXY Note 6则会带来配置更强的高配版本，比如将装载5.8英寸2K分辨率显示屏，并同样采用RGB像素排列的OLED面板， 所配的处理器则为骁龙823或三星自家的高频猫鼬处理器，并提供6GB RAM的内存容量，电池容量则为4000+毫安时。

不过，针对三星GALAXY Note 6将有曲面屏版本的说法，网友@Rec2020_Eight则表示该机是否有曲面屏或是双版本真的很难说。而在此前，由于三星GALAXY S7 edge在问世后受到了更多的欢迎，所以传出三星GALAXY Note 6在测试直屏和曲屏两种版本的消息，并且据称后者最终被保留的可能性更大。
